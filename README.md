# House Price Prediction

This is my Week 1 internship project on House Price Prediction.

In this project, I used the Housing dataset to predict house prices using machine learning. I first cleaned and explored the data, then trained two models: Linear Regression and Random Forest Regressor. Finally, I compared their performance using MAE, RMSE, and R² Score and created different visualizations to better understand the data.

## Files Included
- analysis.ipynb – Jupyter Notebook containing the complete code
- Housing.csv – Dataset used for the project
- summary.pdf – Summary of the project and findings
- charts/ – Images of the generated charts

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Overall, this project helped me improve my understanding of data preprocessing, model training, model evaluation, and data visualization using Python.
